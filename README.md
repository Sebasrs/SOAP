# SOAP
This is a project for the SOA4ID course in the Computer Engineering carrer in the Tecnol√≥gico de Costa Rica.
It is used to measure the liquid soap and gel alcohol levels in a bathroom. It sends this information to a mobile app to be monitored by the user.
This project uses the following technologies:
* ReactNative
* NodeJS
* Express
* MongoDB
* CloudMQTT
* NodeMCU module

## Installation

You need to install NodeJS and NPM in your computer

Run the following commands in /Front-end and /Back-end:
To install the node dependencies
```sh
npm install
```

To install the expo package
```sh
npm install -g expo-cli
```

To run the web server:
run the following command on /Back-end
```sh
node index.js
```

To run the react native:
run the following command on /Front-End
```sh
expo start
```
